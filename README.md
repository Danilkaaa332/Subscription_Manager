## Как запустить

1. Скачайте проект и откройте в Visual Studio
2. Нажмите "Восстановить" когда появится запрос о NuGet пакетах  
3. Нажмите F5 или "Запуск" для сборки и запуска

## Что нужно для работы

- Visual Studio
- Доступ в интернет для загрузки пакетов

## Ответы на вопросы

**1. Работа с датами**  
Использовал свойство IsUpcoming, которое сравнивает дату списания с текущей датой. Учитывал, что DateTime содержит время, поэтому использовал Date для сравнения только дат.

**2. Расчеты стоимости**  
Логика преобразования периодов находится в классе Subscription в свойствах MonthlyCost и YearlyCost. Выбрал это место, потому что расчеты это часть данных подписки и используются в разных частях приложения.

**3. Валидация URL**  
Сейчас URL отображается как текст. Можно улучшить через Hyperlink в WPF для кликабельных ссылок.

**4. Архитектура**  
Добавил бы отдельный сервис NotificationService, который бы периодически проверял подписки с приближающимися списаниями используя тот же алгоритм что и IsUpcoming.
